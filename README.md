# 🚛 Truckonomy
[🔗 Open Truckonomy](https://nikunjsaini007.github.io/truckonomy/)

**Truckonomy** is a lightweight dashboard designed for players of **Euro Truck Simulator 2 (ETS2)** and **American Truck Simulator (ATS)** who want a more **immersive trucking company experience**.  

The idea behind this project is simple: while playing truck simulators, I wanted a **realistic way to manage my own trucking business** - tracking trucks, drivers, finances, and completed jobs - in a **clean, simple, interactive dashboard**.  

With Truckonomy, you can log deliveries, monitor company growth, and manage your fleet in a **simple, organized interface**, all running directly in the browser with **no backend required**.

---

## ✨ Features

- 🏢 **Company Dashboard**: Save and display your company name dynamically.  
- 💰 **Finance Tracking**: Monitor total money, profit, loss, and company level.  
- 🚚 **Fleet Management**: Add and manage trucks, assign drivers, track locations.  
- 👨‍✈️ **Driver Management**: Add drivers, track levels and earnings, remove when needed.  
- 📦 **Job History**: Log delivery jobs with cargo, route, distance, and profit.  
- 🔄 **Automatic Profit Updates**: Jobs automatically update company finances.  
- 💾 **Persistent Data**: All data saved using LocalStorage, so it stays after page refresh.  
- 🎨 **Clean UI**: Interactive particle background and glow effects for a modern dashboard look.

Truckonomy was created to enhance the **immersive experience of running a trucking company** while playing ETS2 or ATS.
